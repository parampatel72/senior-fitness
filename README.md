# Senior Fitness App

A modern, accessible fitness application designed specifically for older adults (aged 55+). The app features a clean, intuitive interface with large, touch-friendly buttons and high-contrast design elements.

## Tech Stack

- React + TypeScript
- Tailwind CSS
- Framer Motion
- React Router
- Heroicons

## Design Principles

- **Accessibility First**: Large text (min. 16pt), high contrast, and clear visual hierarchy
- **Senior-Friendly**: Touch-friendly buttons, simple navigation, and clear instructions
- **Calming Aesthetics**: Soft pastel gradients and gentle animations
- **Safety Focus**: Emergency support and health monitoring features

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
